# 水声工程 RAG 系统全面测试问题集

## 1. 基础概念与定义 (验证知识库基础)
这些问题测试系统对领域核心概念的掌握程度。
- **什么是声纳方程？** (核心概念)
- **什么是水声工程？** (学科定义)
- **简述水声信道的主要特点。** (领域常识)
- **什么是被动声呐？它与主动声呐有什么区别？** (对比分析)
- **什么是海洋环境噪声？** (环境认知)

## 2. 文档细节检索 (验证查准率)
这些问题测试系统能否从上传的文档（如 `2233.docx`）中精准提取特定信息。
- **人员与机构**:
  - **师国峰的性别是什么？** (特定事实)
  - **殷敬伟在项目中主要负责什么？** (角色识别)
  - **生雪莉、高明生是哪个单位的？** (实体关系)
  - **哈尔滨工程大学在本项目中承担了什么任务？** (机构作用)
- **项目背景与痛点**:
  - **为什么说水声工程专业的实验成本较高？** (因果关系 - 对应文档 "1.1 受实验仪器设备...")
  - **虚拟仿真实验主要解决了本科教学中的哪些难题（桎梏）？** (问题解决 - 对应文档 "4.2 科研反哺...")
  - **该虚拟仿真实验的设计思路是什么？** (对应文档 "2.1.2 设计思路")
- **具体技术参数**:
  - **什么是“清劲风”海况？对应几级海况？** (对应文档图表/描述)
  - **实验中如何体现“国防特色”属性？** (对应文档关键词)

## 3. 综合理解与推理 (验证长文本能力)
这些问题需要系统阅读多个段落并总结，而不仅仅是关键词匹配。
- **请总结虚拟仿真技术在水声工程教学中的主要优势。** (归纳总结)
- **结合文档内容，详细说明“科研反哺教学”是如何实施的？** (机制解释)
- **在复杂海洋环境中，影响声纳被动探测性能的主要因素有哪些？** (综合分析)
- **本项目是如何实现人才培养与科学研究紧密结合的？** (宏观理解)

## 4. 多轮对话测试 (验证上下文记忆)
测试系统能否在连续对话中保持上下文。
- **场景 A (人物追踪)**:
  1. 问：**师国峰是谁？**
  2. 问：**他是哪个学校的？**
  3. 问：**他参与了什么项目？**
- **场景 B (概念深入)**:
  1. 问：**什么是声纳方程？**
  2. 问：**这个方程主要用于解决什么问题？**
  3. 问：**其中有哪些参数与海洋环境有关？**

## 5. 抗干扰与边界测试 (验证鲁棒性)
测试系统在面对无关或错误信息时的表现。
- **幻觉测试**:
  - **水声工程中如何应用火箭发动机技术？** (文档未提及，应回答不知道)
  - **师国峰是否获得过诺贝尔奖？** (事实性否定)
- **无关干扰**:
  - **今天天气怎么样？** (闲聊，应礼貌拒答或仅回答与文档无关)
  - **`dsjfklsdjflksdjfklsdj`** (乱码，应优雅处理)
